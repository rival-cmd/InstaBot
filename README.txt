
This program only requires the modification of the config.ini file.
You will be required to place your username and password in this file,
and you will have to add the list of tag name you want to spider, along with 
updating the file paths to Instabot_Log.txt
(This is where you get details of posts and follows)
and Chrome_Log.txt(Selenium Debugging).
Also, you will have to update or install Google Chrome, and download the
Selenium Webdriver that is compatable with your ChromeVerison Number:
	https://chromedriver.chromium.org/downloads

Finally, you must update the config.ini file with the Path to chromedriver.exe and to the chromeuserdata folder
 (This is where the webdriver will store your session cookies, 
	you can clear cookies by deleting the folders contents between sessions)
